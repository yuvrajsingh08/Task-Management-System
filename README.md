# Task Management System

A dashboard-based Task Management System to streamline project and task management for managers and employees.

## Key Features

- **Project Management**: 
  - Managers can create, view, and manage projects.
- **Task Management**:
  - Managers can add tasks to projects and assign them to employees.
- **Employee Management**:
  - Add and update employee details, including their roles and information.
- **Assignment**:
  - Tasks can be assigned directly to specific employees.
- **Dashboard View**:
  - Central interface to track projects, tasks, and assignments.
  - Overview of project status and employee workloads.
- **User Roles**:
  - Distinction between manager and employee capabilities.

## Technology

- **Language**: JavaScript (99.4%)

- Managers log in to add employee details, create projects, add tasks, and assign them.
- Employees log in to view assigned tasks and update status.
